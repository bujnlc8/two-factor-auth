A simple CLI to generate two-factor-auth token compatible with Google Authenticator.

USAGE:

```

  $ 2fa V3ZZAK3PZ6A3L4LV6JXRVBVW3WOX4EV2 [--no-loop]
  435767  (15 second(s) remaining) ==>  586621

```
